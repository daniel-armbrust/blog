Introdução Algumas das muitas definições existentes para o termo Cloud Computing: Cloud Computing é um modelo computacional no qual é possível provisionar recursos de TI sob demanda, de acordo com o seu negócio e pagar somente pelo uso que se faz. Em sua essência, é uma nova forma de se provisionar recursos e serviços de TI.
